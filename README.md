These segy files contain the data collected at the NASA Marshall Space Flight Center, Huntsville, Alabama, USA.
Data were colelcted at the Lunar Regolith Terrain field on December 8, 2022 (Site 1), July 17, 2023 (Site 1),
and July 18, 2023 (Site 2).
At Site 1 the data were collected both using a traditional geophone array (2022) as wells as using 
wheel-based accelerometers on a prototype rover (2023)
At Site 2, only wheel-based accelerometers on a prototype rover were deployed (2023).
